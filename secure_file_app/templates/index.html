Welcome to the Secure File Storage

Secure File Storage allows Premium users to keep important documents at their fingertips without having to worry about losing them or someone getting unwanted access to them.

Secure File Storage uses the same powerful encryption and security infrastructure protecting your passwords and Secure Notes to store documents in your Dashlane app.

Wherever you have access to your devices, you'll have access to those files. And because Dashlane backs up your encrypted data, you never have to worry about losing your files.
